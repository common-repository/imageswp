Generates free images using ipic.ai and saves them to the WP Media Library in one click.

== Description ==

Free Ai Image Generator in WordPress! Create 100% unique Ai Images inside Wordpress - Free Ai Photo Generator. Works with the Media Library, Gutenberg, and other popular page builders. Just grab a free api key and generate ai images

Note: this plugin makes use of ipic.ai's free API. You can get your FREE API key on [ipic.ai website](https://www.ipic.ai/).

== Installation ==

1) Install imageswp plugin to your blog and activate it.
2) Register on [ipic.ai website](https://www.ipic.ai/)
3) Grab your api key on: [free api key](https://www.ipic.ai/user/settings/api-key)
4) Add the API key to your imageswp plugin.
5) Start generating images related to your blog post.

ENJOY!
